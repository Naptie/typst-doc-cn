    计算 函数族

    数值计算和处理。

    这些函数是 calc 模块的一部分，不会默认导入。除了下面列出的函数外，calc 模块还定义了常数 pi、tau、e、inf 和 nan。

    函数族

    abs

    Calculates the absolute value of a numeric value.

    calc.abs(
    intfloatlengthangleratiofraction
    ) -> any
    #calc.abs(-5) \
#calc.abs(5pt - 2cm) \
#calc.abs(2fr)

    value
    int float length angle ratio fraction
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The value whose absolute value to calculate.

    pow

    Raises a value to some exponent.

    calc.pow(
    intfloat, intfloat,
    ) -> intfloat
    #calc.pow(2, 3)

    base
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The base of the power.

    exponent
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The exponent of the power.

    exp

    Raises a value to some exponent of e.

    calc.exp(
    intfloat
    ) -> float
    #calc.exp(1)

    exponent
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The exponent of the power.

    sqrt

    Extracts the square root of a number.

    calc.sqrt(
    intfloat
    ) -> float
    #calc.sqrt(16) \
#calc.sqrt(2.5)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number whose square root to calculate. Must be non-negative.

    sin

    Calculates the sine of an angle.

    When called with an integer or a float, they will be interpreted as radians.

    calc.sin(
    intfloatangle
    ) -> float
    #assert(calc.sin(90deg) == calc.sin(-270deg))
#calc.sin(1.5) \
#calc.sin(90deg)

    angle
    int float angle
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The angle whose sine to calculate.

    log

    Calculates the logarithm of a number.

    If the base is not specified, the logarithm is calculated in base 10.

    calc.log(
    intfloat, base: float,
    ) -> float
    #calc.log(100)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number whose logarithm to calculate. Must be strictly positive.

    base
    float

    The base of the logarithm. May not be zero.

    默认：10.0

    ln

    Calculates the natural logarithm of a number.

    calc.ln(
    intfloat
    ) -> float
    #calc.ln(calc.e)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number whose logarithm to calculate. Must be strictly positive.

    The number whose factorial to calculate. Must be non-negative.

    perm

    calc.perm(
    int, int,
    ) -> int
    #calc.perm(10, 5)

    base

    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number of permutations. Must be non-negative.

    #calc.binom(10, 5)

    n

    必需参数位置参数

    k
    int
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The lower coefficient. Must be non-negative.

    int
    必需参数位置参数

    The second integer.

    floor

    Rounds a number down to the nearest integer.

    If the number is already an integer, it is returned unchanged.

    calc.floor(
    intfloat
    ) -> int
    #assert(calc.floor(3.14) == 3)
#assert(calc.floor(3) == 3)
#calc.floor(500.1)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to round down.

    ceil

    Rounds a number up to the nearest integer.

    If the number is already an integer, it is returned unchanged.

    calc.ceil(
    intfloat
    ) -> int
    #assert(calc.ceil(3.14) == 4)
#assert(calc.ceil(3) == 3)
#calc.ceil(500.1)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to round up.

    trunc

    Returns the integer part of a number.

    If the number is already an integer, it is returned unchanged.

    calc.trunc(
    intfloat
    ) -> int
    #assert(calc.trunc(3) == 3)
#assert(calc.trunc(-3.7) == -3)
#assert(calc.trunc(15.9) == 15)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to truncate.

    fract

    Returns the fractional part of a number.

    If the number is an integer, returns 0.

    calc.fract(
    intfloat
    ) -> intfloat
    #assert(calc.fract(3) == 0)
#calc.fract(-3.1)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to truncate.

    round

    Rounds a number to the nearest integer.

    Optionally, a number of decimal places can be specified.

    calc.round(
    intfloat, digits: int,
    ) -> intfloat
    #assert(calc.round(3.14) == 3)
#assert(calc.round(3.5) == 4)
#calc.round(3.1415, digits: 2)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to round.

    digits
    int

    The number of decimal places.

    默认：0

    clamp

    Clamps a number between a minimum and maximum value.

    calc.clamp(
    intfloat, intfloat, intfloat,
    ) -> intfloat
    #assert(calc.clamp(5, 0, 10) == 5)
#assert(calc.clamp(5, 6, 10) == 6)
#calc.clamp(5, 0, 4)

    value
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The number to clamp.

    min
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The inclusive minimum value.

    max
    int float
    必需参数位置参数
    位置参数按顺序传入，不带名称。

    The inclusive maximum value.

    可变参数
    可变参数可以多次传入。

    The sequence of values from which to extract the minimum. Must not be empty.
