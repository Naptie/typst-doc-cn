    model

    文档结构。

    在这里，您可以找到用于构建文档结构并与该结构进行交互的函数。包括章节标题、图表、文献管理、交叉引用等内容。

    定义

    • bibliography文献目录 / 引用列表。
    • cite引用文献或元素。
    • document文档的根元素及其元数据。
    • emph通过切换至斜体强调内容。
    • enum有序列表。
    • figure带有可选描述的图表。
    • footnote脚注。
    • heading章节标题。
    • link链接到 URL 或文档中的位置。
    • list无序列表。
    • numbering对有序的事物应用编号。
    • outline目录、图表或其他元素的大纲。
    • par将文本、间距和内联级元素排列成段落。
    • parbreak段落分隔。
    • quote显示引用，带有一系列可选属性。
    • ref对标签或文献的引用。
    • strong通过增加字重强调内容。
    • table表格。
    • terms术语列表。
