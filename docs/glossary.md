# 术语表
以下是翻译时所使用的中英对照术语表，以及相应的可能存在的相应注解。


| 英文       | 中文                     | 详情链接                                   |
| ---------- | ------------------------ | ------------------------------------------ |
| content    | 内容 / 内容块            | 根据上下文的情况，可能被翻译成内容或内容块 |
| set rule   | set 规则                 | 出现于 [样式]($styling) 部分               |
| show rule  | show 规则                | 出现于 [样式]($styling) 部分               |
